syntax high lighterの使い方

syntax high lighterとは？

簡単に言うとhtml上にソースコードをカッコよく表示してくれるプラグイン
今までソースをHTMLで表示させててなんかカッコ悪いなと思って調べてみた。

  • オフィシャルサイト
  • オフィシャルサイト ダウンロード先

埋め込み方法

  1. 1.ダウンロードしてくると何やらたくさん落ちてくるが、ひとまず「script」配下と「style」配下を使う。
  2. 2.適当なディレクトリに上記の2ファイル配下を配置
  3. 3.一緒にダウンロードされてきた「index.html」のhead内の記述をコピペしてパスを直す。
    「index.html」にあったソースもコピペすると下記のように表示される。 
    function helloSyntaxHighlighter()
{
	return "hi!";
}

「script」配下と「style」配下

「script」配下
「shBrush******.js」のファイルは、それぞれの言語に対応したスクリプト（htmlとかcssとかjsとかphpとかいろいろ）
「style」配下
「shCore.css」は、ハイライトさせるためのコアの処理をする CSSファイルのため、装飾の情報は持っていない
「shTheme******.css」のファイルは、それぞれのテーマに合った装飾（背景や行番号などの色）を設定しているファイル
基本「shCore.css」と「shTheme******.css」をセットで使うが、あらかじめセットになっているのが「shCore******.css」
  • オフィシャルサイト 各テーマ

お試し

  • 1.HTMLを表示させてみる。
    HTMLを表示させるときはpreタグに「class="brush: html;"」をつける

  • 2.CSSを表示させてみる。
    CSSを表示させるときはpreタグに「class="brush: css;"」をつける

    .wrapper .linkList {
  margin-top: 30px;
  display: flex;
  flex-wrap: wrap;
  justify-content: space-around;
}
.wrapper .linkList li {
  width: 80%;
  padding: 0 10px;
  border: 1px solid #d2d2d2;
  margin: 10px 0 0;
  text-align: left;
}
.wrapper .linkList li:hover {
  background: #eee;
}
.wrapper .linkList li a {
  display: block;
  padding: 10px;
}
.wrapper .linkList li a::hover {
  text-decoration: underline;
}
.wrapper .linkList li a::before {
  content: "";
  display: inline-block;
  width: 0;
  height: 0;
  margin-right: 10px;
  border-style: solid;
  border-width: 5px 0 5px 8px;
  border-color: transparent transparent transparent #ff601c;
}

  • 3.JSを表示させてみる。
    JSを表示させるときはpreタグに「class="brush: js;"」をつける

    function helloSyntaxHighlighter()
{
	return "hi!";
}

基本的にはこれでつかえる！

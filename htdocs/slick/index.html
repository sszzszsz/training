ふつうの画像単体のスライダー

「slick-theme.css」で左右のボタンなどのcssを読み込んでいる（この中でwebフォントも読み込みに行っているので、これを読み込むと重くなる）が大抵カスタマイズするので読み込まなくてもいい。必要な記述を移植するのがベター。
デフォルトだと矢印などの背景が白なので、スライダーの背景が白だと有るのか無いのかわからないので注意。
上のスライドは左右のボタンの位置をcssで上書いている。

$(function() {
  $('.slickSample1').slick();
});

画像単体の自動スライダー

カスタマイズについて詳しく書いてあるサイト

  • どこよりも詳しい万能スライダーjQueryプラグインslick.jsの使い方

下のスライダーは自動再生とスライダーの下にドットのボタンを表示させるように設定。

$(function() {
  $('.slickSample2').slick({
    autoplay: true,
    autoplaySpeed: 2000,
    dots: true,
    speed: 1000
  });
});

画像複数の自動スライダー

以下のスライダーは左右に余白を指定 + フェードで切り替わるように設定（7行目）。

$(function() {
  $('.slickSample3').slick({
    autoplay: true,
    autoplaySpeed: 3000,
    dots: true,
    speed: 3000,
    fade: true,
    infinite: true,
    slidesToShow: 3,
    slidesToScroll: 3,
    responsive: [
      {
        breakpoint: 768,
        settings: {
          slidesToShow: 1,
          slidesToScroll: 1,
          infinite: true,
          dots: true
        }
      }
    ]
  });
});

サムネイル付きのスライダー

$(function() {
  var $slider_container = $('.slider-container'),
    $slider = $('.slickSample4'),
    $slider_nav_container = $('.slider-nav-container'),
    $slider_nav = $('.slider-nav');
    // ナビゲーション用に複製
    $slider_nav.append($slider.contents().clone());
    
    // スライド初期化時にクラスを追加
    // はじめはdisplay:noneしておき、.initializedが追加されたらdisplay:block
    $slider.on('init', function(){
      $slider_container.addClass('initialized');
    });
    $slider_nav.on('init', function(){
      $slider_nav_container.addClass('initialized');
    });
    
    $slider.slick({
      arrows: false,
      asNavFor: $slider_nav,
      fade: true,
      waitForAnimate: false
    });
    $slider_nav.slick({
      appendArrows: $slider_nav_container,
      slidesToShow: 3,
      asNavFor: $slider,
      focusOnSelect: true,
      centerMode: true,
      centerPadding: '40px'
    });
    // スライドの横幅を小数点以下pxまで表示
    // 今回はcenterPaddingもあるのでその分も計算しています
    $slider_nav.on('setPosition', function(){
      var slider_width = $slider_nav.width(),
      slide_gutter = $slider_nav.find('.slick-slide').eq(0).css('margin-right').split('px')[0],
      slides_num = $slider_nav.slick('slickGetOption', 'slidesToShow'),
      slides_center_padding = $slider_nav.slick('slickGetOption', 'centerPadding').split('px')[0],
      slide_width = (slider_width - slide_gutter * (slides_num - 1) - (slides_center_padding * 2)) / slides_num;
      
      $slider_nav.find('.slick-slide').css('width', slide_width + 'px');
    });
  });

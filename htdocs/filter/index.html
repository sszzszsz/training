css filterについて

css filterとは？

photoshopの画像加工（明るさとか彩度とかそういうの）がcssでも出来る。CSSのSVG仕様の一部で、写真やイラストなどの画像や文字にフィルタ（効果）を重ねて画像の加工をする機能

ただし！！！IEだとcss filterは使えません！！！

このページもIE11で見るとただただ同じ画像が並んでるだけのページになってます。

彩度：saturate
0%～上限なしを指定でき、色の鮮やかさを変更することが出来る。
上限が無いので1000％とか指定も可能。
0にすると彩度がなくなってく(白黒になってく)
明度：brightness
0～上限なしをを指定でき、色の明るさを変更することが出来る。指定に単位は付けない。
上限が無いので1000とか指定も可能。
0にすると暗くなっていき（黒くなる？）、大きくなるにつれて明るくなってく
色相：hue-rotate
0deg～360deg（上限・下限なし）を指定でき、色相を調整できる。
指定が角度なのは色相環で計算されるから（懐かしい。。）
色相環について
コントラスト：contrast
0%～上限なしを指定でき、コントラストを調整できる。
上限が無いので1000％とか指定も可能。
100%だと元画像と変わらない(？)が、だんだん数を小さくするとコントラストが弱くなっていく
グレースケール：grayscale
0～100%を指定でき、100%に近づくにつれてだんだん白黒になってく。
下の例は80%で指定してみた。
セピア：sepia
0～100%を指定でき、100%に近づくにつれてだんだんセピア色になってく。
下の例は80%で指定してみた。
余談:セピア色の「セピア」って何の意味？まさに色々な色の語源
階調の反転：invert
0～100%を指定でき、値が大きいほど、写真のネガっぽくなる。
ぼかし：blur
0px～上限なしを指定でき、値が大きいほど、ぼかしの度合いが大きくなる。値の上限が無く、例えば1000pxという値も設定可能。
10px以上指定するのはもうぼやぼやしすぎてみづらすぎる。。下の例は5pxを指定。
ドロップシャドウ：drop-shadow
drop-shadow(値1 値2 値3 rgba(値4,値5,値6,値7)で指定。
  • 値1：横方向への影。下限上限なし
  • 値2：縦方向への影。下限上限なし
  • 値3：影のぼかし具合。下限上限なし
  • 値4～6：色の指定する（色の三原色rgb）0～255
  • 値7：明るさを指定する。0～上限なし
